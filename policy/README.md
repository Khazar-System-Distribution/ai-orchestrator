# policy — Təhlükəsizlik qaydaları mühərriki

- Zero Trust arxitekturasının əsas komponentidir.
- Hər sorğu agentə çatmadan icazə yoxlamasından keçir.
- Default olaraq bütün sorğular rədd edilir (deny by default).
- Yalnız açıq şəkildə icazə verilmiş action/agent/capability cütlüklərinə allow verilir.
